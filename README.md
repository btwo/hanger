# 自用的Web骨架

包括了一些常用的设置，函数等等等等，能帮助快速开发。基于Tornado Web框架。

## 使用
没有文档，需要熟悉Tornado Web框架。

软件假定你使用的是Linux系统，WebServer是Nginx，如果有其他需求请自行修改。

## 安装需求
* python 2.7
* tornado
* wtforms
* jinja2
* markdown2
* elixir
* Python Image library(PIL)
